リンクの指定

リンクの色を指定する

「<body 各種色指定>」と「</body>」の間にリンクの指定指示を書く。
リンクの指定は次項以降をよく見る。
以下の表を参考に色を指定していく。
bgcolor="色"背景色を指定
text="色"文字色を指定
link="色"通常のリンク文字の色
alink="色"マウスでクリックした後のリンクの文字色
vlink="色"訪問済みのリンクの文字色


リンク

文字をリンクにする
文字をクリックすると、別のページに飛ぶ。

<a href="リンク先のアドレス">サンプルページ</a>


画像をリンクにする
画像をクリックすると、別のページに飛ぶ。

<a href="リンク先のアドレス"><img src="画像ファイルのパス" border="0"></a>



文字をリンクにする（新しいウィンドウを開く）
文字をクリックすると、別のページを新しいウィンドウで開く。

<a href="リンク先のアドレス" target="_blank"> サンプルページ </a>


画像をリンクにする（新しいウィンドウを開く）
画像をクリックすると、別のページを新しいウィンドウで開く。

<a href="リンク先のアドレス" target="_blank"><img src="画像ファイルのパス" border="0"></a>



同じページ内でリンクを張る
<a href="#タイトル">クリックすると同じページ内のどこかに移動する</a>
上記のリンクをクリックするとHTMLのコードにおいて、「<a name="タイトル">」と書かれている部分に移動する。 例として、このページのタイトル部分（一番上）に移動するように設定した。
